Quick start

Clone, build, and run. The following commands assume you have a C++ toolchain and HiGHS available (or installed via your package manager).

git clone https://github.com/Asif-Iqbal-Bhatti/Ultra-fast-Hull-Energy-via-LP.git
cd Ultra-fast-Hull-Energy-via-LP
# build instructions (example)
mkdir build && cd build
cmake ..
cmake --build . --config Release

About

This repository implements a very fast method to compute the energy hull (Ehull) using linear programming solvers such as HiGHS. Use this page as a quick link hub to the repo and instructions.

  • Author: Asif Iqbal Bhatti
  • Language: C++ (HiGHS-based LP)
  • Purpose: fast computation of Ehull for materials/energy analysis

Usage example

After building, run the executable (example name used here: ehull) passing input data as required by the repo's tools.

// example
./ehull --input examples/sample_data.json --output results/ehull.json
